# Tavern - iOS Application

Welcome to the GitHub repository for the Tavern iOS application. Tavern is designed to bring the experience of [TavernofAzoth.com](http://TavernofAzoth.com) to your iOS device.


### Prerequisites

- Xcode 15.0 or later
- iOS 16.0 or later
- CocoaPods
